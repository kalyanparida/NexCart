# NexCart 

A Django-based eCommerce web application.

## Features
- Product listing
- Add to cart
- Order placement
- Order tracking (status)
- Admin panel

## Tech Stack
- Python (Django)
- HTML, CSS, JavaScript
- SQLite

## How to Run
1. Clone repo
2. Run `python manage.py runserver`
3. Open http://127.0.0.1:8000/
